# v2.0 실기기 테스트 체크리스트

> 대상 브랜치: `release/v2.0`
> 코드 리뷰 수정 + Riverpod 마이그레이션 + 테스트 확대 + 접근성 + 보안 수정 반영

---

## 1. 기본 플로우

- [ ] 콜드 스타트 → 스플래시 → 홈 화면 정상 진입
- [ ] 로그인 (이메일/비밀번호)
- [ ] 로그인 (Google OAuth)
- [ ] 로그인 (Apple OAuth)
- [ ] 로그인 (Kakao OAuth)
- [ ] 회원가입 → 프로필 설정 → 펫 등록 → 홈
- [ ] 로그아웃 → 재로그인 → 이전 데이터 정상 표시

## 2. Riverpod SSOT — 펫 전환 (핵심)

- [ ] 홈 탭에서 펫 칩 탭 → 프로필 화면 → 다른 펫 선택
- [ ] 펫 전환 후 홈 탭 돌아왔을 때 BHI/건강 요약 자동 갱신
- [ ] 펫 전환 후 체중 탭 돌아왔을 때 차트 자동 갱신
- [ ] 펫 전환 후 AI 백과사전 탭 진입 시 올바른 펫 표시
- [ ] 펫 추가 → 홈에 즉시 반영
- [ ] 펫 삭제 → 다른 펫으로 자동 전환

## 3. 데이터 기록 (오프라인 동기화)

- [ ] 체중 기록 저장 → 차트에 즉시 반영
- [ ] 같은 날짜 체중 다중 기록 → 모두 표시
- [ ] 음식 기록 저장 → 성공
- [ ] 수분 기록 저장 → 성공
- [ ] 비행기 모드에서 기록 → 오프라인 큐 저장
- [ ] 비행기 모드 해제 → 자동 동기화 (앱 resume 시)

## 4. 건강 체크 (AI Vision)

- [ ] 건강 체크 메인 → 모드 선택 (전체/부위별/배변/사료)
- [ ] 카메라 캡처 → 분석 화면 → 결과 표시
- [ ] 결과 저장 → 히스토리에 표시
- [ ] 히스토리에서 삭제

## 5. AI 백과사전

- [ ] 질문 입력 → SSE 스트리밍 응답 표시
- [ ] 쿼터 제한 확인 (무료 사용자)
- [ ] 대화 히스토리 유지 (앱 재시작 후)

## 6. 프리미엄

- [ ] 프리미엄 화면 → 플랜 표시
- [ ] 프로모 코드 입력 → 프리미엄 활성화 → 즉시 반영 (Riverpod 연쇄 갱신)
- [ ] 프리미엄 상태 → 쿼터 무제한 확인

## 7. 프로필/설정

- [ ] 프로필 편집 (이름, 사진)
- [ ] 펫 프로필 편집 (이름, 종, 성별, 생일)
- [ ] 언어 변경 (한국어/영어/중국어) → 즉시 반영
- [ ] 소셜 계정 연동/해제
- [ ] 회원 탈퇴 → 데이터 삭제 확인

## 8. 보안 수정 확인

- [ ] Rate limiting: 로그인 10회 초과 → 429 응답
- [ ] Rate limiting: 회원가입 5회 초과 → 429 응답
- [ ] 비밀번호 재설정: 6자리 코드 발송
- [ ] 비밀번호 재설정: 5회 잘못 입력 → 코드 무효화
- [ ] 비밀번호 규칙: 8자 미만 → 422 에러
- [ ] 타인 펫 데이터 접근 시도 → 403 (IDOR 수정 확인)

## 9. 접근성 (VoiceOver/TalkBack)

- [ ] VoiceOver 켜고 홈 화면 탐색 → 모든 버튼 읽힘
- [ ] 펫 선택 카드 → "Select pet, button" 읽힘
- [ ] 뒤로 가기 버튼 → "Go back, button" 읽힘
- [ ] 체크박스 → 체크 상태 읽힘

## 10. 엣지 케이스

- [ ] 네트워크 끊김 상태에서 앱 사용 → 크래시 없음
- [ ] 서버 다운 시 → 로컬 캐시 폴백 동작
- [ ] 펫 0마리 상태에서 홈 화면 → 크래시 없음
- [ ] 긴 펫 이름 (20자+) → UI 깨지지 않음
- [ ] 앱 백그라운드 → 포그라운드 복귀 → 정상 동작
