# 온보딩 화면 UI 개선

**날짜**: 2026-01-05
**파일**:

- [lib/src/screens/onboarding/onboarding_screen.dart](../../lib/src/screens/onboarding/onboarding_screen.dart)

## 구현 목표

- 온보딩 화면의 시각적 복잡도를 낮추고 단일 페이지로 간소화.
- 맨 바깥쪽 큰 원환을 제거하여 더 깔끔한 디자인 구현.
- 텍스트 콘텐츠가 원환과 겹치지 않도록 레이아웃 조정.
- 페이지 인디케이터 제거로 단일 온보딩 페이지로 전환.

## 주요 변경 사항

### 1. 배경 원환 구조 변경

- **기존**: 3개의 동심원 구조 (Ellipse_120, Ellipse_69, Ellipse_68)
- **변경**: 2개의 동심원 구조 (Ellipse_69, Ellipse_68)
- 가장 바깥쪽 큰 원환(Ellipse_120.svg) 제거로 시각적 복잡도 감소
- `_buildBackgroundCircles` 함수에서 `largeRingScale`, `largeRingSize` 관련 코드 제거

### 2. 하단 콘텐츠 레이아웃 조정

- **위치 변경**: `bottom: 0` → `bottom: h(48)`
- 텍스트("만나서 반가워요!" 및 설명)가 원환과 겹치지 않도록 위치 조정
- 하단 여백 중복 제거 (`SizedBox(height: h(48))` 제거)
- 설명 텍스트와 버튼 사이 간격 조정: `h(32)` → `h(48)`

### 3. 페이지 인디케이터 제거

- 3개의 점으로 표시되던 페이지 슬라이스 인디케이터 완전 제거
- `_buildPageIndicator` 함수 삭제 (미사용 함수 제거)
- 온보딩을 다중 페이지에서 단일 페이지로 변경
- "시작하기" 버튼 클릭 시 바로 로그인 화면으로 이동

## UI 개선 효과

- 더 깔끔하고 단순한 첫 화면 제공
- 텍스트 가독성 향상 (원환과의 겹침 제거)
- 사용자 경험 간소화 (단일 페이지 온보딩)
- 불필요한 UI 요소 제거로 집중도 향상

## 기술적 세부사항

- 반응형 레이아웃 유지 (`w()`, `h()` 함수 활용)
- 디자인 기준 해상도: 393x852
- 새, 나무, 나뭇잎 등 메인 콘텐츠는 변경 없이 유지
- 그라데이션 원 및 중앙 일러스트레이션 유지
