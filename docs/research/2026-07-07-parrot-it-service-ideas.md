# 앵무새 관련 IT 서비스 아이디어 11선

> 2026-07-07 브레인스토밍. perch_care(앵무새 건강관리 앱)를 기반으로, 꼭 이 앱의 기능이 아니어도 되는 앵무새 관련 IT 서비스 아이디어를 발굴·브리프화한 문서.
> 1~10번은 IT 서비스, 11번은 실물 굿즈 사업(깃털 기념품) + 한국 실행 로드맵.

## 전제: perch_care 현재 자산

- Flutter 앱 + FastAPI/PostgreSQL 백엔드 (Railway 배포)
- AI 비전 건강체크, BHI/WCI 건강지수, 체중·먹이·물 기록, AI 백과사전(LLM)
- 오프라인 동기화(SyncService), FCM 푸시, 다국어(ko/en/zh), 펫 프로필
- 이 자산의 재사용 가능성이 각 아이디어의 "시너지" 항목에 반영됨

## 1. 발성(소리) 분석 서비스

| 항목 | 내용 |
|------|------|
| 타겟 | 하루 종일 새 곁에 있을 수 없는 보호자, 특히 조류 병원 없는 비수도권 거주자 |
| 문제 | 앵무새는 질병을 은폐해 증상이 보이면 이미 위중 — 호흡음·발성 변화가 몇 안 되는 조기 신호인데 초보자는 감지 불가 |
| 핵심 기능 | ① 10~30초 녹음으로 호흡기 이상음(쌕쌕거림·클릭음) 위험도 스코어링 ② 스트레스 콜/발정 콜 구분 ③ 평소 발성량 베이스라인 대비 급감 알림 |
| MVP | 수동 녹음 → 3클래스 분류(정상/호흡기 의심/스트레스) → BHI 반영 + 조류 병원 안내 |
| 수익 | 월 N회 무료 + 상시 모니터링·발성 히스토리 리포트 구독 (기존 서버 쿼터 인프라 재사용) |
| 시너지 | 펫 프로필·BHI·FastAPI 쿼터·FCM 그대로 재사용, 발성 스코어를 BHI의 새 신호 축으로 통합 |
| 리스크 | 라벨링된 조류 병리 오디오 데이터셋이 사실상 없음 + 오탐·미탐 시 의료 판단 오도 책임 |

## 2. 조류 전문 수의사 원격 상담 플랫폼

| 항목 | 내용 |
|------|------|
| 타겟 | 건강 기록은 쌓고 있지만 근처에 조류 진료 병원이 없는 국내 앵무새 보호자 |
| 문제 | 조류 진료 병원이 전국 수십 곳 수준으로 희소 — 골든타임 내 전문 수의사 판단을 받기 어려움 |
| 핵심 기능 | ① 조류 진료 병원 지도·필터(응급/입원/전문)·예약 ② 화상·채팅 원격 상담(긴급도 트리아지) ③ 체중·BHI·AI 건강체크 기록 원클릭 진료 차트 공유 |
| MVP | 수기 검증한 병원 디렉토리+지도에 소수 제휴 수의사 비동기 채팅 상담(기록 링크 공유)만 |
| 수익 | 상담 건당 수수료 15~20% + 병원 프리미엄 노출·예약 수수료 |
| 시너지 | 기존 백엔드·JWT 인증·건강 데이터를 그대로 재사용 — "기록 앱 → 진료 차트" 전환 비용 거의 없음. 쌓아온 데이터가 해자가 되는 가장 전략적인 아이디어 |
| 리스크 | 수의사법상 원격 진단·처방의 법적 회색지대 + 조류 수의사 공급 절대 부족으로 플랫폼 성립 실패 가능 |

## 3. 실종 앵무새 찾기 네트워크

| 수익 | 찾기 자체는 무료(네트워크 효과 필수), 구독 번들·인증 뱃지·용품 제휴 광고로 간접 수익화 |

| 리스크 | 실종은 저빈도 이벤트 — 평시 사용자·제보자 밀도를 못 유지하면 네트워크가 죽는 콜드스타트 |

| 핵심 기능 | ① 조류 안전 체크리스트(테프론 프리·방충·타 반려동물 격리) 기반 시터 인증·검색·예약 ② 위탁 중 급여·배변 사진·체중 실시간 전송 + BHI 이상 알림 ③ 개체별 케어 지시서(사료·수면시간·금지 접촉) 자동 전달 |

| 수익 | 예약 건당 15~20% 중개 수수료 + 시터 인증 갱신비 + 위탁 중 건강 모니터링 프리미엄 |
| 시너지 | 펫 프로필·기록 스키마·BHI를 위탁일지에 재사용 — 시터 입력 데이터가 보호자의 기존 건강 타임라인에 끊김 없이 연결 |
| 리스크 | 위탁 중 폐사·사고 시 책임·배상 문제(고가·급사 위험·병원 희소) + 검증된 조류 시터 공급 콜드스타트 |

## 11. 앵무새 깃털 굿즈 (깃털 반지·기념품) — 한국 실행 로드맵

> 출발점: 샤오홍슈(rednote) 게시물 — 해시태그 #鹦鹉羽毛(앵무새 깃털) #戒指(반지) #宠物毛发纪念品(반려동물 털 기념품). "죽은 반려동물 애도"가 아니라 **살아있는 내 새의 환우(털갈이) 깃털로 만드는 셀프기프트(悦己 소비)** 포지셔닝이 핵심.

### 11.1 중국에서 검증된 공식

- **공법:** 레진(에폭시/UV) 봉입 + 925 순은 세팅이 표준. 드라이플라워·금박과 함께 밀봉, 제작 15영업일 수준. 최고가 라인은 깃털 탄소로 기르는 기념 다이아몬드(3,400위안~)까지 존재
- **가격대:** 인기 제품(달 펜던트·설산 반지) 380~700위안, 풀커스텀 500위안~, 키링 199위안. 해외는 US$62~165
- **주문 방식:** 고객이 동전 크기 분량의 깃털을 공방에 등기우편 발송 → 제작 → 배송. 상담은 메신저 기반 — **앱 내 주문·추적으로 IT화하기 쉬운 구조**
- **인기 제품군:** 목걸이/펜던트 > 반지 > 키링·팔찌. 인기 상품 월 판매 ~50건, 기념 메달류 누적 12만 건+
- **시장 성장:** 중국 반려동물 장례·기념 시장 2018년 7억 → 2023년 18.25억 위안(5년 2.6배), 반려인의 25%가 기념품 제작 선택. 중국 앵무새 시장 자체가 5년간 40억→100억 위안, 샤오홍슈 #鹦鹉 조회 41억 회

### 11.2 한국 시장 상황

- **직접 경쟁자 없음:** 개·고양이 털 주얼리 공방은 다수 존재(멍냥공방 실버 목걸이 5만원, 앤로펫 털 반지 5.2만~7.8만원, 14k 61.5만원)하고 인스타 공방 '리버러플'이 앵무새 깃털 목걸이 제작 사례가 있으나 — 모두 개·고양이 중심의 부수 옵션. **앵무새/조류 특화 브랜드는 확인되지 않는 빈 니치**
- **시장 규모:** 국내 반려조류 사육 약 4만~6만 가구(반려가구의 0.7~1%). 소수지만 앵무새 분양가 100만~200만원 수준으로 1인당 지출 여력 높음. 최대 커뮤니티는 네이버 카페 '앵사모'
- **펫로스 시장 순풍:** 반려가구 54.7%가 펫로스 경험, '유골을 보석으로' 이용 의향 33.6%(20대 최고), 1인당 장례 소비 11만→26만원 증가 — 다만 이 사업은 펫로스 **이전**(살아있는 새)의 빈 포지션을 노림
- **채널 수수료:** 아이디어스 15%+무료배송 부담 vs 네이버 스마트스토어 약 6% vs 인스타 DM 무수수료 — **스마트스토어 + 인스타 DM 조합이 유리**

### 11.3 법률 체크 (교차 검증 완료 — 이 사업의 첫 관문)

**핵심 분기점: 어떤 종의 깃털을 받느냐로 규제가 갈린다.**

| 구분 | 내용 |
|------|------|
| ✅ 규제 없음 | **CITES 부속서 제외 4종(학명 기준): 사랑앵무(Melopsittacus undulatus)·왕관앵무(Nymphicus hollandicus)·목도리앵무(Psittacula krameri)·모란앵무(Agapornis roseicollis)** — 국제적 멸종위기종이 아니므로 야생생물법 제16조 미적용, 일반 판매 요건만 지키면 됨. 국내 최다 사육종이라 시장성도 충분 |
| ⚠️ 학명 주의 | 제외는 위 4개 학명에만 적용 — **피셔모란앵무(A. fischeri) 등 같은 속의 다른 종은 부속서 II 규제 대상**. "모란앵무는 괜찮다"고 통칭으로 뭉뚱그리면 오판 위험 |
| 🟡 CITES 종(코뉴어·금강앵무 등 부속서 II) | 국내 판매 자체는 수출입 허가 대상 아님. 단 **입수경위 증명서류(수입허가서/인공증식증명서) 보관 의무**(제16조 8항, 미보관 시 과태료 100만원 이하)가 가공품에 명문 적용 |
| 🔴 회색앵무(부속서 I, 2017.1 발효) | 상업적 국제거래 원칙 금지 — 취급 비권장 |
| ❓ 최대 불확실 | 제16조 3항 **용도변경 제한**: 반려 목적 수입 CITES종의 환우 깃털을 상업 판매품으로 전용하는 것이 '용도변경'인지 조문상 불명확(위반 시 2년 이하 징역/2천만원 벌금) — **관할 유역·지방환경청 사전 질의 필수** |
| 일반 요건 | 반복 판매 시 사업자등록 필수. 통신판매업 신고는 연 50회 미만/간이과세자면 면제이나 오픈마켓 입점엔 신고번호 요구가 많아 신고 권장. 국내 판매는 검역 비대상(수출 시엔 검역+CITES 재수출허가 동시 발생) |

### 11.4 운영 모델

- **진입장벽 낮음:** 레진공예 원데이 클래스 약 30만원, "배우기 3개월 만에 판매 개시" 사례 존재. 반지 베이스 부자재 개당 400원 수준
- **단가 구조(반지 5.5만원 기준):** 재료비 ~1만원 + 왕복 등기·포장 ~8천원 + 수수료 → **노동 제외 마진 약 3만원(마진율 ~55%)**
- **주문 플로우:** 주문 → 깃털 등기 수령(2~3일) → 제작(1~2주) → 배송 = **총 2~3주**
- **리스크 방어 표준:** 여분 깃털 2~3장 의무 접수 + 수령 즉시 사진 공유 + 약관에 분실·재작업 책임 한도 명시
- **시즌성:** 환우는 봄·가을 집중이지만 "빠진 깃털을 보관해뒀다 주문"하는 패턴이라 판매는 연중 분산 가능
- **IT 연계(차별화 핵심):** perch_care에 '환우/깃털 기록' 추가 → 환우 시즌 FCM 푸시("깃털 보관하세요") → 주문 딥링크(perchcare:// 스킴). **펫 프로필의 품종 필드로 CITES 제외 4종 여부를 자동 판별해 접수 가능/불가를 사전 안내 — 법적 리스크 차단과 UX를 동시에 해결**

### 11.5 실행 로드맵

#### Phase 0 — 법률 확정·수요 검증 (4~6주, 약 50~70만원)
**목표:** 돈 쓰기 전에 법적 리스크를 0으로, 실수요 20건+를 문서로 확보
1. 관할 유역·지방환경청 서면 질의 — 제16조 3항 용도변경 해당 여부 + 6항 양도신고의 가공품 적용 여부
2. 접수 정책 확정 — CITES 제외 4종 '학명 기준' 한정 접수 (동속 타종 차단 문구 포함)
3. 레진공예 원데이 클래스 수강 → 자기 새 깃털로 시제품 3~5개 + 촬영 샘플
4. 앵사모·앵무새 갤러리·인스타에 시제품 공개 + 사전주문 의향 설문(5.5만~8만원대 수용도)
5. 앤로펫·멍냥공방·리버러플 벤치마크표 + 원가·마진 시뮬레이션(목표 마진율 50%+)

**성공 기준:** 환경청 회신(또는 4종 한정으로 질의 불요 확인) + 사전주문 의향 20~30건

#### Phase 1 — MVP 판매, 수작업 공방 검증 (2~3개월, 약 150~250만원)
**목표:** 앱 없이 수작업만으로 첫 20~30건 유료 판매 + 운영 SOP 완성
1. 사업자등록 + 통신판매업 신고
2. 스마트스토어(6%) + 인스타 DM 채널 개설 — 아이디어스(15%)는 보류
3. 텀블벅 펀딩 또는 앵사모 공동구매로 사전주문 20~30건 접수·제작·배송 (반지 5.5만원대, 납기 2~3주 고지)
4. 주문 SOP: 여분 깃털 의무 접수·수령 사진 공유·책임 한도 약관·종(학명) 확인 체크리스트
5. 제작 과정 릴스·후기 콘텐츠 축적 (샤오홍슈식 '살아있는 내 새의 셀프기프트' 톤)

**성공 기준:** 유료 20~30건 완납·배송, 재작업률 10% 미만, 실측 마진율 50%+, 후기 10건+

#### Phase 2 — perch_care 앱 연동, IT 해자 구축 (2~3개월, Phase 1과 병행, 현금 50만원 이하)
**목표:** 환우 기록 → 깃털 보관 알림 → 주문 딥링크 퍼널을 앱에 내장
1. 펫 프로필·캘린더·BHI에 '환우/깃털 기록' 기능 추가 (환우기 건강 추적과 자연 연결)
2. 품종 필드 기반 CITES 자동 판별 — 제외 4종만 주문 가능 안내
3. 주문 폼 + perchcare:// 딥링크 + 주문·배송 추적 화면 (FastAPI에 주문 엔드포인트 추가)
4. 환우 시즌(봄·가을) FCM 푸시 → 주문 전환 퍼널 계측(Analytics 이벤트)
5. 기존 사용자 앱 내 배너·첫 주문 할인 전환 테스트

**성공 기준:** 앱 경유 주문 30%+ 또는 월 10건+, 환우 기록 → 주문 전환율 측정 가능

#### Phase 3 — 확장: 제품군·CITES종·해외 (3~6개월, 약 300~500만원)
**목표:** 월 20~50건 규모화 + 고단가 라인·해외 판로 타당성 확정
1. 제품 라인 확장 — 펜던트·키링·팔찌(중국 인기 순위 반영), 14k 고단가 옵션
2. CITES 부속서 II종(코뉴어 등) 접수 개시 검토 — Phase 0 환경청 회신 전제, 고객 증빙서류 확보·보관 프로세스 구축
3. 제작 병목 해소 — 세공 외주 파트너 또는 주형 표준화로 납기 2주 유지
4. 해외 진출 사전 검토 — 깃털 국제배송의 CITES 수출허가 + 검역 요건 (허가 확보 전 Etsy·중국 역진출 착수 금지)
5. 재구매·선물 수요 검증 + 구독형 '환우 시즌 키트' 실험

**성공 기준:** 월 매출 200~300만원 안정화, 납기 준수율 90%+, 확장 Go/No-Go 근거 문서화

### 11.6 Go/No-Go 기준

> **Phase 0 종료 시점**에 ① "CITES 제외 4종 학명 한정 접수로 법적 리스크 사실상 없음" 확인 + ② 유료 전환 가능한 사전주문 의향 20건 이상 — 둘 다 충족하면 Go, 하나라도 미달이면 No-Go(또는 정책 재설계).

**요약:** 중국에서 검증된 공식(우편 깃털 수령 → 레진+925은 → 380~700위안대)은 소자본으로 즉시 복제 가능하고, 한국은 앵무새 특화 경쟁자가 없는 빈 니치. CITES 제외 4종 한정 접수로 법적 리스크를 원천 차단하고, perch_care의 환우 기록·품종 자동 판별·주문 딥링크로 타겟 정합성 100%의 자체 주문 채널이라는 진입장벽을 만드는 것이 최소 리스크 경로. 초기 현금 투입 약 200~300만원 수준.
