# 캐러셀 이미지 카피 (8장)

小红书 캐러셀은 첫 3장이 노출에 가장 큰 영향을 줍니다. 커버 → 차별화 훅 → 핵심 기능 순으로 무게 배분.

> ⚠️ 멀티모달/食物安全/4가지 분석 모드 미배포 → 슬라이드에 포함하지 않음.

---

## 디자인 공통 가이드

- **브랜드 컬러:** `#FF9A42` (오렌지 Primary), `#FF7C2A` (서브 강조)
- **배경:** 흰색 or 연한 크림, 텍스트 가독성 우선
- **타이포:** 헤드 굵은 산세리프 (중국어: 思源黑体 Bold)
- **로고 배치:** 우상단 or 우하단 고정, 16pt 이하로 작게
- **여백:** 상하좌우 최소 48px 확보, 텍스트 집중도 ↑
- **캐러셀 비율:** 3:4 (세로형, 小红书 최적)

## 이미지별 카피

### 1. 커버 (가장 중요)

**메인:** `鹦鹉健康记录 2.0`
**서브:** `你们的反馈，做成了这次更新 🦜`
**디자인 노트:**
- 숫자 `2.0`을 크게, 오렌지 컬러로 강조
- 뒷배경에 흐린 앱 UI 실루엣 or 앵무새 일러스트
- 우하단에 앱 아이콘 작게

---

### 2. 건강 리포트 공유 훅 (⭐ 2.0 핵심 차별화)

**메인:** `🔗 一个链接`
**서브 (크게):** `把健康记录分享给兽医`
**하단 배지:** `看诊前准备 · 不用下载 app`
**디자인 노트:**
- 실제 건강 리포트 공유 URL → 브라우저 렌더 화면 스크린샷 (더미 데이터)
- 화면은 체중 그래프 + BHI 점수 + 요약 텍스트가 보이게
- 스마트폰 프레임 안에 웹 화면 — "웹 링크로 공유된다"는 느낌 시각화
- 2월 댓글 "我正有需要记录我的鹦鹉给医生参考" 원문 인용 (상단 작게)

---

### 3. 건강 요약 카드 + 주간 인사이트

**메인:** `每周 AI 生成`
**서브 (크게):** `健康变化报告`
**하단 설명:** `BHI · 体重 · 异常次数 · 饮食规律 一目了然`
**디자인 노트:**
- 홈 화면의 건강 요약 카드 스크린샷 (中文 UI)
- 우측에 FCM 푸시 알림 미리보기 말풍선 ("本周鹦鹉洞察报告已生成 🦜")
- 화살표로 카드 → 상세 인사이트 화면 플로우 표시

---

### 4. Before / After (피드백 대응 가장 임팩트)

**헤드:** `你们说的，我们改了`
**좌측 라벨:** `2月 · 以前` (약관 한국어 스크린샷)
**우측 라벨:** `2.0 · 现在` (약관 중국어 스크린샷)
**하단 강조:** `中文 100% 本地化`
**디자인 노트:**
- 좌우 2분할, 가운데에 화살표
- 각 스크린샷에 빨간 펜으로 "韩语" / 초록 펜으로 "中文" 표시

---

### 5. BHI 건강지수 시각화

**메인:** `BHI 健康指数`
**서브:** `0-100 分 · 一眼看懂小鸟状态`
**하단 설명:** `体重 60% + 饲料 25% + 水分 15%`
**디자인 노트:**
- BHI 게이지 화면 스크린샷 사용
- 점수 영역별 색상 분포 (90+ 녹색, 60-89 노랑, ~59 빨강)
- 주간 차트 미리보기 동반

---

### 6. 피드백 대응 표 (인포그래픽)

**헤드:** `2月的反馈 vs 2.0的修复`
**표 내용 (6개 row):**
- 约款只有韩语 → ✅ 中文本地化
- 时间没有中文 → ✅ 国际化格式
- 体重删不掉 → ✅ 删除修复
- 保存失败 → ✅ 离线同步
- app 卡顿 → ✅ 快 1~3 秒
- 密码没确认 → ✅ 6 位验证码

**디자인 노트:**
- 좌측 빨간색 박스 (불만) / 우측 초록색 박스 (해결)
- 화살표로 대응 관계 명시
- 하단에 `… 还有更多细节在 app 里等你发现 ✨`

---

### 7. 为爱发电 배너 (감사 메시지)

**메인:** `目前 全部 AI 功能`
**서브 (크게):** `免费开放`
**하단:** `作为感谢 ❤️`
**디자인 노트:**
- 가장 단순하고 강렬한 레이아웃
- 오렌지 그라데이션 배경 or 크림 배경 + 큰 하트 일러스트
- `全部` 강조 (글자 크기, 굵기)
- 내부 메모: `目前` 수식어로 향후 유료화 여지 보존

---

### 8. 안드로이드 테스터 모집

**메인:** `招募 · 安卓内测员 🤖`
**서브:** `评论留言 or 私信 = 内测名额`
**하단 가이드:**
```
1. 评论区留【鸟品种 + 手机型号】
2. 或私信 Google 账号邮箱
```
**디자인 노트:**
- 안드로이드 공식 로봇 로고 사용 금지 → 자체 일러스트 재해석 권장 + 초록 포인트
- "名额有限，先到先得" 작게 추가 (긴급성)
- 우하단에 화살표 → 프로필로 유도

## 영상 대체안 (선택)

캐러셀 2~3번을 **15초 세로형 숏폼 영상**으로 교체 시 참여도 상승 기대:

1. **0~3초:** 2.0 커버 + "你们的反馈做成了更新" 자막
2. **3~8초:** 건강 리포트 공유 플로우 — app 에서 링크 생성 → 웹 뷰 열기 → 수의사 노트북 화면
3. **8~12초:** 홈 화면 건강 요약 카드 + 주간 AI 인사이트 알림
4. **12~15초:** Before/After 약관 스크린샷 스윕 + "中文 100%" CTA

영상 사용 시 캐러셀은 4~8장으로 축소.
