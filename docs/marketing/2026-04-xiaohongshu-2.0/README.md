## v2.0 실제 셀링 포인트 (배포됨)

### 🆕 신규 기능

1. **📋 건강 변화 요약 카드** (홈 화면)
   - BHI / 체중 / 이상소견 카운트 / 급여·음수 일관성 한눈에

2. **🤖 주간 건강 인사이트** (GPT 기반)
   - 매주 AI가 주간 데이터 분석 → 요약 + 추천사항
   - 다국어 (ko/en/zh), FCM 푸시 알림

3. **🔗 건강 리포트 & 병원 방문 요약 웹 링크 공유** ⭐
   - JWT 토큰 기반 공유 URL (7일 유효)
   - **수의사 방문 전 링크 한 번으로 기록 전달** (앱 설치 불필요)
   - 2월 댓글 "我正有需要记录我的鹦鹉给医生参考" 니즈 직접 대응

4. **🦜 AI 咨询师 "鹦鹉博士" 업그레이드**
   - 챗봇+비전 에이전트 정확도 23건 개선
   - 중국어 이름 통일 (智能助手 → 鹦鹉博士)
   - 언어 혼용/마크다운 포맷 개선

### 🛠️ 2월 피드백 반영

| 피드백 | 2.0 대응 |
|---|---|
| 약관/FAQ 한국어만 | 中文 100% 본지화 |
| 시간 표시 중국어 없음 | 날짜/시간 포맷 국제화 |
| 체중/펫 삭제 불가 | 삭제 기능 수정 |
| 저장 실패 | 오프라인 큐 + 자동 동기화 |
| 앱 렉 | 시작 속도 1~3초 단축 |
| 비밀번호 확인 단계 없음 | 6자리 인증 + 재검증 제한 |

> ⚠️ **식품 안전 조회**(能不能查询鹦鹉能不能吃这个食物) 피드백은 멀티모달 브랜치 기능이라 **포스트에서 언급 금지**. 댓글로 물어오면 "향후 버전 검토 중"으로 간결 응대.
