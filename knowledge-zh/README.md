# 鹦鹉养护知识库（中国版）

## 简介

本知识库是 perch_care 应用程序的中国本地化知识资源，专为中国鹦鹉饲养者量身打造。所有内容均基于中国法律法规、市场环境、气候条件及文化背景编写，旨在为用户提供准确、实用、合规的鹦鹉养护信息。

知识库共包含 **46 篇文档**，涵盖 **8 大分类**，覆盖鹦鹉饲养的方方面面。

## 文档结构说明

- 所有文档采用统一的 Markdown 格式，使用 H2/H3 标题进行内容分块
- 文档结构经过优化，适配 RAG（检索增强生成）管线的向量化切分
- 每篇文档控制在 80-130 行，确保语义完整且检索效率高
- 文件名使用英文命名（URL 安全），内容全部为简体中文
- 各文档末尾均附有参考来源，确保信息可溯源

## 知识分类

### 1. 法律法规 (legal/)

中国鹦鹉饲养相关法律法规汇编。包括《野生动物保护法》解读、人工繁育许可证办理流程、合法饲养品种清单、各省市地方性法规差异等内容。帮助饲养者在合法合规的前提下享受养鸟乐趣。

**核心文档**：合法品种清单、许可证办理指南、违规处罚标准、跨省运输规定

### 2. 品种指南 (species/)

中国市场上常见且合法饲养的鹦鹉品种详细介绍。涵盖虎皮鹦鹉、玄凤鹦鹉、牡丹鹦鹉、小太阳鹦鹉等热门品种的外观特征、性格特点、饲养难度评级、寿命预期及适合人群分析。

**核心文档**：各品种详情页、品种对比表、新手推荐品种、颜色变异图鉴

### 3. 营养与饮食 (nutrition/)

基于中国市场可获取食材的鹦鹉营养指南。包括主粮配比建议、中国本土水果蔬菜安全清单、滋养丸品牌对比、自制鸟粮配方、季节性饮食调整方案及常见饮食误区纠正。

**核心文档**：安全食物清单、有毒食物警示、滋养丸选购指南、四季饮食方案

### 4. 健康与疾病 (health/)

鹦鹉常见疾病的预防、识别与治疗指南。涵盖禽类疾病症状速查表、中国各城市禽类宠物医院推荐、居家急救措施、定期体检项目建议及中国常见传染病防控要点。

**核心文档**：疾病症状速查、禽类医院名录、居家急救手册、体检项目指南

### 5. 饲养管理 (care/)

针对中国不同气候区域的鹦鹉日常饲养管理方案。包括温湿度调控、光照管理、笼具布置、卫生清洁规范、换羽期护理、繁殖管理以及北方供暖季和南方梅雨季的特殊应对措施。

**核心文档**：日常护理规范、季节管理指南、换羽期护理、繁殖管理手册

### 6. 训练指南 (training/)

科学系统的鹦鹉行为训练方法。涵盖基础上手训练、语言模仿教学、飞行召回训练、社会化培养、问题行为矫正（如咬人、尖叫、拔羽）及正向强化训练原则详解。

**核心文档**：上手训练教程、语言训练方法、行为问题矫正、社会化培养指南

### 7. 养鸟文化 (culture/)

中国悠久的养鸟文化传承与现代社区资源。包括中国传统养鸟历史、现代鹦鹉社区（贴吧、微信群、抖音博主）、线下鸟友聚会信息、花鸟市场导览及鹦鹉摄影分享文化。

**核心文档**：养鸟文化简史、线上社区导航、花鸟市场导览、鸟友活动指南

### 8. 市场指南 (market/)

中国鹦鹉市场购买渠道与用品选购指南。涵盖各品种价格参考、线上线下购买渠道对比、笼具食具等必备用品选购建议、主流电商平台商品甄选技巧及避坑指南。

**核心文档**：品种价格参考、用品选购指南、购买渠道对比、防骗避坑手册
