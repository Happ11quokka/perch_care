# 鹦鹉紧急救护指南

本指南涵盖鹦鹉饲养中可能遇到的紧急情况及急救处理方法。请提前阅读并熟悉操作流程，紧急时刻才能从容应对。

---

## 急救总原则

1. **保持冷静**：慌乱会加重鸟的应激反应
2. **保温优先**：受伤或生病的鸟体温下降很快，立即保温至30-32℃
3. **安静环境**：将鸟移至安静、光线柔和的环境中
4. **记录症状**：拍照/录像记录伤情，便于兽医判断
5. **尽快就医**：急救只是临时措施，不能替代专业治疗

---

## 出血（血羽断裂）

### 什么是血羽

血羽是正在生长中的羽毛，内部有血管供给营养。断裂后会持续出血，如不处理可能导致失血过多甚至死亡。

### 急救步骤

1. 用干净毛巾轻柔包裹鸟身，固定住鸟（注意不要压迫胸部影响呼吸）
2. 找到出血的血羽位置
3. 用玉米淀粉或面粉按压止血点，持续按压3-5分钟
4. 如按压止血无效，需用止血钳或尖嘴镊子顺着羽毛生长方向将断裂血羽完整拔出
5. 拔出后继续按压止血1-2分钟
6. 止血后将鸟放回笼中安静休息，密切观察30分钟

### 注意事项

- 拔血羽需要一定技巧，操作前确保握稳鸟身
- 不要用剪刀剪断血羽，残留的羽管会持续出血
- 止血粉（云南白药粉剂）也可用于辅助止血
- 出血量大或无法止血时立即就医

## 中毒

### 特氟龙（不粘锅）中毒

- **来源**：不粘锅、电热锅、烤箱、电烫斗、吹风机等含聚四氟乙烯（PTFE）涂层的器具在高温加热时释放有毒气体
- **症状**：突然呼吸困难、跌落栖杆、抽搐、死亡（往往极其迅速）
- **急救**：立即将鸟转移到通风良好的房间，关掉所有含PTFE的加热器具；如鸟仍有呼吸，提供新鲜空气并立即就医
- **预防**：**最佳做法是将含PTFE/不粘涂层器具移出有鸟家庭**，改用不锈钢、铸铁、玻璃等替代材质。厨房严禁养鸟；仅“关门关窗”不足以避免风险

### 食物中毒

- **危险食物**：巧克力、咖啡因、酒精、鳄梨（牛油果）、洋葱、大蒜、含盐过多的食物
- **症状**：呕吐、腹泻、嗜睡、抽搐、失去平衡
- **急救**：确认误食物品，移除毒物来源，立即就医并告知兽医误食物品名称和大致量

### 植物中毒

- **常见有毒植物**：百合、水仙、夹竹桃、万年青、滴水观音、龟背竹
- **症状**：口腔肿胀、流涎、呕吐、腹泻
- **急救**：移除植物残渣，用清水轻轻冲洗口腔（如鸟配合），立即就医

### 金属中毒

- **来源**：含铅/锌的玩具、首饰、钥匙、铅笔、老式窗框涂料等
- **症状**：嗜睡、呕吐、绿色粪便、抽搐、共济失调
- **急救**：移除可疑金属物品，立即就医，X光检查可确认是否有金属异物在体内

## 骨折

### 判断方法

- 翅膀下垂无法收拢（翼骨骨折）
- 腿部无法负重、异常弯曲（腿骨骨折）
- 触摸时鸟表现剧烈疼痛

### 急救步骤

1. 将鸟轻柔移至小型容器（如铺软布的鞋盒）中，限制活动
2. **不要**自行尝试复位或固定骨折部位
3. 移除笼中所有栖杆，将食水容器放在底部便于进食
4. 保温，尽快就医
5. 运输途中保持平稳，减少颠簸

### 注意事项

- 鸟类骨骼中空，骨折后愈合需要专业固定
- 不建议非专业人员自行包扎，错误操作可能造成二次伤害

## 中暑

### 高危场景

- 夏季将鸟笼放在阳光直射处
- 密闭车内（即使短暂停留也极度危险）
- 高温高湿环境下通风不良

### 症状

- 张嘴呼吸、喘气
- 翅膀张开远离身体
- 站立不稳、嗜睡
- 严重时抽搐、意识丧失

### 急救步骤

1. 立即转移至阴凉通风处
2. 用常温水（不是冰水）轻轻喷洒脚部和翅膀下方
3. 可用湿毛巾轻覆脚部帮助降温
4. 提供新鲜饮水（不要强迫灌水）
5. 稳定后尽快就医

### 注意事项

- 切勿用冰水降温，温差过大会导致休克
- 中国南方夏季湿热环境下尤需注意，室温超过35℃须采取降温措施

## 惊夜症（夜间惊恐）

### 什么是惊夜症

鸟在夜间突然惊醒，在笼中剧烈扑腾，可能撞伤自己。玄凤鹦鹉（鸡尾鹦鹉）尤其多发。

### 可能诱因

- 窗外突然的灯光（汽车大灯、闪电）
- 噪音（鞭炮声、雷声 -- 春节期间尤为常见）
- 笼中有害虫（螨虫等）刺激
- 其他宠物（猫）靠近鸟笼

### 急救步骤

1. 立即开一盏小灯（不要突然开强光），让鸟看到周围环境
2. 用平静柔和的声音安抚，不要急于用手去抓
3. 等鸟安静后，检查是否有受伤（出血、翅膀异常等）
4. 如有出血按"出血急救"流程处理

### 预防措施

- 夜间保持一盏微弱夜灯
- 春节/重大节日期间注意隔音，关好门窗
- 窗帘拉好，避免外部灯光突然照射
- 定期检查笼具，排除害虫

## 溺水

### 常见场景

- 鸟飞入马桶、水槽、鱼缸、汤锅等
- 洗澡时水量过深

### 急救步骤

1. 立即将鸟从水中捞出
2. 保持头颈自然位置并轻柔清理口鼻可见水液/污物；**不要倒置甩水或剧烈摇晃**
3. 用干净柔软的毛巾轻轻吸干身上的水分
4. 立即保温（可使用加热灯或暖贴包裹的毛巾）
5. 如呼吸停止或极弱，立即联系急诊兽医并按电话指导处理；**不建议无培训者自行胸部按压**
6. 尽快就医

### 预防

- 放飞时盖好马桶盖、厨房无明水容器
- 洗澡用浅水盘，水深不超过鸟腿部一半

---

## 电击

### 常见场景

- 鸟啃咬电线/充电线

### 急救步骤

1. 先关闭电源，不要直接用手接触触电中的鸟
2. 检查口腔和脚部是否有灼伤
3. 保温，保持安静
4. 立即就医

### 预防

- 所有电线使用保护套管
- 放飞活动时收好所有裸露电线和充电器

---

## 卡蛋/难产

### 什么是卡蛋

母鸟因各种原因无法正常排出体内的蛋，属于危及生命的紧急情况。

### 症状

- 蹲在笼底，频繁用力但排不出蛋
- 腹部明显膨大
- 呼吸急促
- 瘫痪或站立困难（蛋压迫神经）
- 泄殖腔可能可见蛋的一部分

### 急救步骤

1. 立即保温至30-32℃（温度对于卡蛋母鸟至关重要）
2. 提高环境湿度（在笼旁放一碗热水增加蒸汽）
3. 在泄殖腔周围涂抹少量食用油或凡士林润滑
4. **不要**用手挤压腹部试图帮助排蛋，可能导致蛋壳破裂造成内伤
5. 以上处理无效须立即就医，兽医可能需要注射催产素或手术取蛋

### 预防

- 保证饮食中充足的钙质（墨鱼骨、矿物质块）
- 避免人为刺激母鸟繁殖（减少筑巢材料和阴暗角落）
- 保证充足光照和运动

## 家庭急救箱清单

建议每位鸟友准备以下急救物品：

| 物品 | 用途 |
|------|------|
| 玉米淀粉/宠物止血粉（经兽医确认） | 止血 |
| 止血钳/尖嘴镊子 | 拔除断裂血羽 |
| 干净毛巾 | 固定鸟身、保温 |
| 纱布/棉签 | 清洁伤口 |
| 碘伏（稀释） | 伤口消毒 |
| 电解质粉 | 补充体液 |
| 保温灯/暖贴 | 保温 |
| 食用油/凡士林 | 卡蛋润滑 |
| 小型运输盒 | 紧急就医运输 |
| 兽医联系电话 | 紧急联系 |
