# 日常健康检查指南

鹦鹉是善于隐藏病情的动物，在野外表现出虚弱意味着被天敌捕食。因此，日常细心观察至关重要——很多疾病在早期通过仔细观察是可以发现的。

---

## 每日观察清单

每天花5-10分钟对鸟进行以下观察，建议在固定时间（如早晨喂食时）进行：

### 眼睛

- **正常**：明亮有神、清澈、左右对称
- **异常信号**：眼睛浑浊、肿胀、有分泌物、频繁眨眼、眼周脱毛或红肿

### 鼻孔/鼻蜡膜

- **正常**：鼻孔干净通畅、鼻蜡膜光滑（颜色因品种而异）
- **异常信号**：鼻孔有分泌物、鼻孔堵塞、鼻蜡膜增厚粗糙或变色、打喷嚏频繁

### 喙部

- **正常**：表面光滑、上下喙对齐、长度适当
- **异常信号**：喙部过长或变形、表面有裂纹或分层、出现白色蜂窝状结痂（疥螨）、喙色异常

### 羽毛

- **正常**：羽毛紧贴身体、有光泽、完整
- **异常信号**：
  - 长时间蓬松炸毛（保温行为，提示可能生病）
  - 羽毛脱落异常（非正常换羽期）
  - 啄羽导致的裸露皮肤
  - 羽毛变色、变脆、出现应力纹

### 粪便

- **正常**：成形的深绿色/棕色粪便+白色尿酸盐+少量透明液体尿液
- **异常信号**：详见下方「粪便分析指南」

### 行为与精神状态

- **正常**：活泼好动、对环境有反应、正常进食和饮水、正常鸣叫
- **异常信号**：
  - 嗜睡、闭眼站在栖杆上
  - 蹲在笼底不愿上栖杆
  - 食欲明显下降或停止进食
  - 沉默不叫或叫声异常
  - 尾巴上下摆动（呼吸费力的表现）
  - 头部反复点头或摇晃

### 体态

- **正常**：站姿端正、双脚均匀负重、能自如飞行/攀爬
- **异常信号**：单脚站立（偏向一侧可能提示腿部问题）、翅膀下垂、身体倾斜

## 每周检查项目

每周抽出一次时间进行更细致的检查：

### 体重测量

- 使用厨房电子秤（精度0.1g）每周固定时间称重
- 早晨进食前称重最准确
- 记录体重数据，建立体重曲线

| 常见品种 | 正常体重范围 |
|----------|-------------|
| 虎皮鹦鹉 | 30-40g |
| 玄凤鹦鹉 | 80-120g |
| 小太阳鹦鹉 | 60-80g |
| 和尚鹦鹉 | 90-120g |
| 灰鹦鹉 | 400-550g |
| 亚马逊鹦鹉 | 350-500g |
| 金刚鹦鹉 | 900-1500g |

- **重要**：体重变化超过正常体重的10%须引起警惕，可能提示疾病

### 脚部检查

- 检查脚趾是否有肿胀、结痂、脱皮
- 趾甲长度是否适当（过长需修剪）
- 脚底是否有发红或溃疡（脚垫炎/bumblefoot）
- 栖杆材质和粗细是否合适（不应全部使用同一粗细的栖杆）

### 笼具检查

- 检查笼具是否有生锈或破损
- 食水容器是否清洁
- 玩具是否有破损（可能导致缠绕或误食）
- 栖杆是否稳固

## 每月检查项目

### 喙和趾甲生长

- 正常磨损应保持喙和趾甲在合理长度
- 如需修剪，建议由有经验的人员或兽医操作
- 提供墨鱼骨和矿物质块帮助自然磨喙
- 提供不同质地的栖杆（天然树枝、水泥栖杆等）帮助磨爪

### 胸部肌肉评估

- 轻轻触摸胸骨（龙骨）位置
- 正常状态下胸骨两侧肌肉饱满，龙骨不突出
- 如龙骨明显突出（"刀片状"），提示鸟过瘦，可能有潜在疾病
- 如完全摸不到龙骨，提示鸟过胖

### 食量记录

- 观察并记录月度食量变化趋势
- 食量突然增加或减少都需要关注

## 年度体检建议

建议每年至少带鸟到专业异宠医院进行一次全面体检，内容包括：

| 检查项目 | 说明 | 参考费用 |
|----------|------|----------|
| 体格检查 | 全身触诊、体重、体态评估 | 100-300元 |
| 粪便检查 | 寄生虫、细菌筛查 | 50-200元 |
| 血液检查 | 血常规+生化（肝肾功能等） | 200-800元 |
| 嗉囊液检查 | 检测细菌/真菌感染 | 100-200元 |
| 衣原体检测 | 鹦鹉热筛查（尤其新鸟） | 150-300元 |

> 新购入的鸟建议在购买后一周内进行首次体检，排除携带传染病的可能。

## 粪便分析指南

粪便是判断鸟类健康状况最重要的日常指标之一。正常粪便由三部分组成：

### 粪便组成

| 成分 | 正常表现 | 来源 |
|------|----------|------|
| 粪便部分 | 深绿色至棕色，成形，管状 | 消化道排出的食物残渣 |
| 尿酸盐 | 白色至淡黄色，糊状 | 肾脏排出的代谢废物 |
| 尿液 | 透明液体，少量 | 肾脏排出的水分 |

### 异常粪便对照表

| 异常表现 | 可能原因 | 紧急程度 |
|----------|----------|----------|
| 全水样便（无成形粪便） | 严重腹泻、肾脏疾病、应激 | 高 -- 24小时内就医 |
| 黑色/柏油样粪便 | 上消化道出血 | 高 -- 立即就医 |
| 鲜红色血便 | 下消化道出血、严重寄生虫感染 | 高 -- 立即就医 |
| 黄色/黄绿色粪便 | 肝脏疾病、衣原体感染 | 中高 -- 尽快就医 |
| 未消化食物颗粒 | 消化功能异常、腺胃扩张症 | 中 -- 尽快就医 |
| 尿酸盐变为黄色/绿色 | 肝脏疾病 | 中高 -- 尽快就医 |
| 尿液量大增（多尿） | 饮水过多、肾脏疾病、糖尿病 | 中 -- 观察并就医 |
| 粪便气味恶臭 | 细菌感染、消化道疾病 | 中 -- 观察并就医 |
| 粪便带有气泡 | 酵母菌/细菌过度生长 | 中 -- 观察并就医 |

### 粪便观察注意事项

- 饮食会影响粪便颜色：吃了大量蔬果后粪便可能偏稀或变色，属于正常现象
- 清晨第一便通常较大且较稀，属于正常
- 母鸟在繁殖期粪便量大且偏稀，属于正常
- 建议使用白色笼底纸，便于观察粪便颜色和性状

## 体重监测的重要性

### 为什么体重监测至关重要

- 鹦鹉的羽毛会掩盖体型变化，肉眼很难发现缓慢消瘦
- 体重变化往往是疾病的最早期信号之一
- 定期称重能建立个体基准线，偏差时及早发现

### 体重监测方法

1. 准备一台精度0.1g的电子秤（厨房秤即可，淘宝/京东购买方便）
2. 在秤上放一个小型栖架或平台，归零
3. 训练鸟站上秤台（可用零食引导）
4. 每周固定时间（建议早晨喂食前）称量并记录
5. 使用Perch Care应用记录数据，自动生成体重趋势图

### 体重异常的判断

| 变化情况 | 可能原因 | 建议操作 |
|----------|----------|----------|
| 单周下降超过5% | 急性疾病、寄生虫、进食困难 | 立即就医 |
| 持续缓慢下降 | 慢性疾病、营养不良、寄生虫 | 尽快就医 |
| 快速增重 | 腹水、肿瘤、肥胖 | 就医排查 |
| 持续缓慢增重 | 饮食过量、运动不足 | 调整饮食和运动 |

## 需要立即就医的危险信号

出现以下任何一种情况，请立即前往异宠医院：

- 呼吸困难或张嘴呼吸
- 持续出血不止
- 抽搐或痉挛
- 意识丧失或无法站立
- 疑似中毒（已知接触有毒物质）
- 持续呕吐
- 泄殖腔脱垂
- 腹部明显膨大伴呼吸急促（疑似卡蛋）
- 24小时以上不进食
- 蹲在笼底、羽毛蓬松、眼睛紧闭

> **提醒**：鸟类表现出明显症状时，病情往往已经较为严重。任何可疑异常都不应"再观察几天"，而应尽早就医。
