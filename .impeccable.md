## Design Context

### Users
- **Target:** 전체 반려조(앵무새 등) 보호자 — 초보부터 경험자까지
- **Context:** 반려조 건강 상태를 데이터 기반으로 관리하고, AI 건강 체크로 전문적인 인사이트를 받는 상황
- **Job to be done:** 반려조의 체중, 식이, 수분 섭취를 기록하고, 건강 이상 징후를 조기에 발견하여 안심하기

### Brand Personality
- **Voice:** 따뜻하고 신뢰감 있는, 데이터 기반의 전문적 케어 파트너
- **Emotional goals:** 안심, 신뢰, 즐거움, 전문성
- **Visual identity:** Warm orange (#FF9A42) 중심의 친근하면서도 깔끔한 톤

### Aesthetic Direction
- **Visual tone:** 따뜻한 오렌지 브랜드 컬러 + 클린 화이트 서피스 + 정돈된 그레이스케일 계층
- **Theme:** Light mode only (다크 모드 미지원)
- **Typography:** Pretendard (한국어 최적화 산세리프)
- **Layout:** 카드 기반, 16px 패딩, 4px 단위 스페이싱 그리드
- **Iconography:** SVG 커스텀 아이콘 + Material Icons 혼용 (통일 필요)

### Design Principles
1. **데이터 신뢰 (Data Trust):** 건강 데이터는 명확하고 읽기 쉽게 — 차트, 수치, 트렌드를 최우선으로 시각화
2. **일관된 경험 (Consistent Experience):** 로딩, 에러, 빈 상태 등 모든 인터랙션 패턴을 앱 전체에서 동일하게 제공
3. **따뜻한 전문성 (Warm Expertise):** 수의학적 전문성을 친근한 톤으로 전달 — 차갑지 않은 헬스케어
4. **최소 마찰 (Minimal Friction):** 기록 입력은 최소한의 탭으로, 결과 확인은 한 눈에
